# Inspire HPC CLI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为 Inspire CLI 增加 `inspire hpc` 命令组，支持高性能计算任务的创建、查看、停止，并提供与现有 `job` 一致的基础体验（缓存、等待、列表、JSON 输出）。

**Architecture:** 采用“独立 `hpc` 命令组 + 独立 OpenAPI 封装 + 独立 sbatch 脚本生成器 + 独立本地缓存文件”的方案，避免复用当前仅面向 GPU 训练任务的 `job --resource` 解析链路。配置层增加 `hpc` 默认项、`hpc.presets.*` 预设和 `workspaces.hpc` 路由，CLI 通过 preset 驱动 `spec_id` / `logic_compute_group_id` / `ntasks` / `cpus_per_task` / `memory_per_cpu` 等 CPU/HPC 参数，并统一生成符合平台要求的 `entrypoint`（sbatch 脚本，最终必须使用 `srun`）。

**Tech Stack:** Python 3.10+, Click, pytest, Inspire OpenAPI client, 现有 config loader / formatter / auth 体系
